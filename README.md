# Movies

### About

Movies is a website that can search the movies you want and see the informations of it.

### Features

- Search Movies
- Show Search Suggestions
- Show Movie's Information
- View All Related Searched Movie

### Technology Used

- ReactJS
- Vercel
- OMDB API
